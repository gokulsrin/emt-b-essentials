## Text

<b>ICS</b> uses {{c1::single command::structure}} — one Incident Commander with overall authority — rather than shared or parallel command, to prevent conflicting orders.

## Extra

Unified Command is the exception: used when multiple agencies have jurisdiction (e.g., state police + fire + EMS), but a single IC is still designated to speak for the group.
