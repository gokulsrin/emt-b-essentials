## Text

<b>Hemorrhage classes</b> by % blood volume lost: Class I {{c1::< 15%}}, Class II {{c1::15–30%}}, Class III {{c1::30–40%}}, Class IV {{c1::> 40%}}.

## Extra

Class I: minimal symptoms. Class II: compensated shock (tachycardia, narrowing pulse pressure). Class III: decompensated (hypotension appears). Class IV: life-threatening; immediate intervention required.
